The Open Source Community is for Everyone

About Us: Open Sesame is a mentorship platform, connecting interested contributors with experienced developers to help newcomers start working on open source projects. Whether you are looking to get started in open source, get new contibutors for your project, or get advice from someone more experienced, Open Sesame can help you make connections within the open source community.

Looking to explore?

Check out the Mentors and Projects pages to browse available opportunities!

Ready to connect?

Once you sign in and create a profile, you can send an introduction email to any mentor on the Mentors or Projects pages.

Want to be a mentor?

Sign in, create a profile, and then you will be able to visit the Dashboard page. On the dashboard, click the 'Become a Mentor' button and input a project you work on to be added to our mentor database.
