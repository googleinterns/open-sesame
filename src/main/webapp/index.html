About Us:

OpenSesame is a mentorship platform, connecting interested contributors with experienced developers to help newcomers start working on projects. Whether you are looking to explore new ventures, get new contibutors onboard, or get advice from someone more experienced, OpenSesame can help you make connections within the open source community.
